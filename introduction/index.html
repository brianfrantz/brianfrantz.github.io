Introduction

29 May, 2014

Since nobody will probably really read this blog, I guess having writer's block is pretty silly. I've been avoiding the blog thing largely for that reason. Hard to believe I'm doing this, but I guess as I've gotten older I've gotten to the point where I have things I want to say.

My name is Brian Frantz. I'm a father of three, a life-long Philadelphia sports fan, and a software developer. I've been working in software for 15 years now (wow - I'm old).

My plan for this site is pretty open ended. I have some thoughts on software development, management, people, processes, etc. that I want to share. If nothing else this is a way to gather my thoughts on each subject.

Hopefully you will find my posts useful and/or thought provoking. I would love to hear some feedback.

My next post will have something more meaty in it. Promise.
